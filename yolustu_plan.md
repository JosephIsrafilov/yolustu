# 🚗 Yolmates — План проекта

> Платформа попутных поездок для Азербайджана (аналог BlaBlaCar)

---

## 1. Обзор продукта

| Параметр | Описание |
|---|---|
| **Название** | Yolmates (Yolmates — «По пути») |
| **Суть** | Веб-платформа с адаптивным мобильным интерфейсом для поиска и предложения попутных поездок по Азербайджану |
| **Целевая аудитория** | Водители и пассажиры, путешествующие между городами Азербайджана (Баку ↔ Гянджа, Баку ↔ Ленкорань, Баку ↔ Шеки и т.д.) |
| **Проблема** | Нет удобного цифрового сервиса для поиска попутчиков. Люди ищут попутки через WhatsApp-группы, Telegram и «сарафанное радио» |
| **Решение** | Централизованная веб-платформа с верификацией документов, рейтингами, онлайн-оплатой Stripe, чатом в реальном времени и ИИ-рекомендациями цен |

---

## 2. Ключевые фичи (MVP)

### 2.1 Для пассажиров
- 🔍 Поиск поездок по координатам (карта) или названию города, дате и количеству мест
- 📍 Указание промежуточных точек посадки/высадки
- 💳 Онлайн-бронирование и оплата картой через Stripe
- ⭐ Рейтинг и отзывы о водителях после завершения поездки
- 💬 Чат в реальном времени с водителем после подтверждения бронирования
- 🔔 Уведомления в системе о статусе бронирования (pending, accepted, rejected, cancelled)

### 2.2 Для водителей
- ➕ Публикация поездки с интерактивной картой (выбор точек сбора и высадки кликом по карте)
- 🧠 Рекомендация оптимальной цены на основе ИИ (NVIDIA NIM LLaMA-3.1-8b-instruct)
- ✅ Подтверждение / отклонение пассажиров в личном кабинете
- 💰 Безопасный прием платежей через Stripe
- ⭐ Рейтинг и отзывы о пассажирах

### 2.3 Общие
- 📱 Регистрация и вход через номер телефона (симуляция SMS OTP с сохранением кода в Redis)
- 🪪 Загрузка документов на верификацию водительского статуса
- 🗺️ Интерактивная карта маршрута (Leaflet Map на базе OpenStreetMap)
- 🌐 Поддержка локализации: азербайджанский язык (основной)

## 4. Технический стек

### 4.1 Frontend (Веб-приложение)
| Компонент | Технология | Обоснование |
|---|---|---|
| **Фреймворк** | Next.js 16 (App Router) | Быстрый SSR, отличная оптимизация под SEO и производительность |
| **Стилизация** | Tailwind CSS v4 | Современный, сверхбыстрый CSS-фреймворк с отличной кастомизацией |
| **Состояние** | Zustand | Простой и быстрый менеджер состояний без лишнего бойлерплейта |
| **Карты** | Leaflet.js (React-Leaflet) | Открытая альтернатива Google Maps, не требующая платных API-ключей |
| **Связь с API** | Axios | Надежный HTTP-клиент с интерцепторами для JWT-токенов |

### 4.2 Backend (API & Сервисы)
| Компонент | Технология | Обоснование |
|---|---|---|
| **Язык & Фреймворк**| Python 3.11 / FastAPI | Асинхронность, встроенная валидация Pydantic, автодокументация OpenAPI |
| **База данных** | PostgreSQL + PostGIS | Надежная реляционная БД с мощными геопространственными запросами |
| **Кэш & Очередь** | Redis | Хранилище сессий, лимитов запросов и временных OTP-кодов |
| **ORM** | SQLAlchemy 2.0 | Современная декларативная ORM с поддержкой асинхронных операций |
| **Миграции** | Alembic | Управление изменениями схемы базы данных |
| **ИИ Интеграция** | NVIDIA NIM SDK | API-доступ к модели `meta/llama-3.1-8b-instruct` для расчета смарт-цен |

### 4.3 Сторонние интеграции
| Сервис | Назначение |
|---|---|
| **Stripe** | Обработка онлайн-оплат (Stripe Checkout) и обработка вебхуков |
| **WebSockets** | Двустороннее real-time соединение для мгновенных сообщений в чате |
| **lsim.az (Симулятор)** | Имитация рассылки СМС-сообщений с OTP кодами на азербайджанские номера |

## 6. Схема базы данных

Реализована схема из 7 ключевых таблиц в PostgreSQL:
- **users**: данные пользователей, их роли, рейтинги и статус модерации документов.
- **device_tokens**: токены устройств для пуш-уведомлений.
- **vehicles**: автопарк водителей (марка, модель, год, номерной знак).
- **rides**: информация о поездках (координаты PostGIS `POINT` для старта/финиша, промежуточные города, цена, свободные места, предпочтения).
- **bookings**: статус бронирования пассажирами мест в поездках.
- **payments**: транзакции Stripe, привязанные к бронированиям.
- **messages**: сообщения во внутригрупповых чатах поездок.
- **reviews**: отзывы пользователей друг о друге по результатам поездки.

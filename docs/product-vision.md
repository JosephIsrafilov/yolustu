# Product Vision — Yolüstü

## Məhsul Vizyonu

Yolüstü, Azərbaycan bazarı üçün hazırlanmış şəhərlərarası carpooling platformasıdır. Tətbiq sərnişinlərə sürücülərlə yol paylaşaraq ucuz, rahat və təhlükəsiz şəkildə şəhərlərarası gediş etmək imkanı yaradır.

**Vizyon cümləsi:**
> "Azərbaycan üzrə ən rahat və əlçatan şəhərlərarası gediş platforması."

## Hədəf Auditoriya

| Seqment | Təsvir |
|---|---|
| Tələbələr | Ucuz gediş axtaran universitetlilər |
| İşçilər | Regionlara tez-tez gedən işçilər |
| Sürücülər | Yanacaq xərclərini bölüşmək istəyən sürücülər |
| Tez-tez səyahətçilər | Şəhərlərarası gedişə ehtiyac duyan insanlar |
| Avtobus/qatar narazıları | İctimai nəqliyyatı narahat tapanlar |

## Problem

Hazırda Azərbaycanda şəhərlərarası gediş axtarışı:
- WhatsApp/Telegram qruplarında xaotik şəkildə aparılır
- Tanınmayan insanlarla yol paylaşmaq risklidir
- Qiymət standartlaşdırılmayıb
- Sürücü haqqında məlumat yoxdur
- Gediş tarixi saxlanmır
- Rəy/reytinq sistemi yoxdur

## Məhsul Dəyəri

| Dəyər | Təsvir |
|---|---|
| Ucuzluq | Şəhərlərarası taksidən ucuz |
| Rahatlıq | WhatsApp qruplarından rahat |
| Sürət | Manuel aranışdan sürətli |
| Lokallaşdırma | Azərbaycan şəhərləri və adi gediş ssenariləri |
| Təhlükəsizlik | Profillər, reytinqlər, rəylər, gediş tarixi |

## Bazar Lokallaşdırması

- Azərbaycan şəhərləri (Bakı, Gəncə, Sumqayıt, Şəki, Quba, Lənkəran, Şamaxı, Mingəçevir, Naftalan)
- Manat (₼) ilə qiymətlər
- Azərbaycan dili interfeysi
- Yerli gediş marşrutları
- Yerli istifadəçi adları
- Yerli maşın modelləri

## Uğur Meyarları

1. İstifadəçi rahat gediş tapa bilir
2. Sürücü asanlıqla gediş elan edə bilir
3. Rezerv prosesi 1 dəqiqədən az çəkir
4. Reytinq sistemi işləyir
5. Admin panel moderasiya imkanı verir

# Technology Stack

## Frontend
- **Framework:** Next.js 16 (TypeScript) - Utilizing the App Router for efficient routing and server components.
- **Styling:** Tailwind CSS v4 - For modern, utility-first styling and responsive design.
- **State Management:** Zustand - A lightweight and flexible state management solution.

## Backend
- **Framework:** FastAPI (Python) - A high-performance, asynchronous web framework for building APIs.
- **ORM:** SQLAlchemy - For flexible and powerful database interactions.
- **Migrations:** Alembic - For managing database schema evolutions.
- **Geospatial Support:** GeoAlchemy2 - To support location-based queries and spatial data.

## Infrastructure
- **Database:** PostgreSQL - A robust and reliable relational database.
- **Cache/Queue:** Redis - Used for caching and potentially for managing asynchronous tasks.
- **Containerization:** Docker & Docker Compose - For consistent local development and deployment environments.
